# Deployment

Deploy MooseStack applications to production environments.

## Overview

MooseStack provides multiple deployment options:
- **Local Development**: Run everything on your laptop
- **Cloud Deployment**: Deploy to cloud platforms
- **Kubernetes**: Container orchestration
- **Docker Compose**: Simple container deployment
- **Offline Deployment**: Air-gapped environments
