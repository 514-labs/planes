## About MooseStack

MooseStack is a type-safe, code-first tooling for building real-time analytical backendsâ€”OLAP Databases, Data Streaming, ETL Workflows, Query APIs, and more.

### Key Features

- **Everything as Code**: Declare all infrastructure in pure TypeScript or Python
- **Hot-reload Development**: Local development environment with instant feedback
- **Pre-integrated Stack**: ClickHouse, Redpanda, Temporal, Redis
- **Type Safety**: Share types across all components
- **Modular Design**: Use only what you need

### Technology Partners

- **ClickHouse**: Online Analytical Processing (OLAP) Database
- **Redpanda**: Streaming platform
- **Temporal**: Workflow orchestration
- **Redis**: Internal state management
